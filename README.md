# 🛍️ Challenge Alura Store

**Análisis de Ventas por Tienda - Challenge de Alura LATAM**

Este proyecto realiza un **análisis exploratorio de datos (EDA)** sobre la información de ventas de **cuatro tiendas** diferentes. El objetivo es identificar insights clave para la toma de decisiones estratégicas, utilizando herramientas de análisis y visualización en Python.

---

## 🧰 Tecnologías Utilizadas

- Python
- pandas
- matplotlib
- seaborn

---

## 📊 Análisis Realizados

- 💰 **Ingresos Totales por Tienda**  
  Comparativa de facturación entre las cuatro tiendas.

- 🗂️ **Ventas por Categoría**  
  Identificación de las categorías más y menos vendidas por tienda.

- 🛒 **Productos Más y Menos Vendidos**  
  Visualización de los productos top y de baja rotación en cada tienda.

- ⭐ **Calificación Promedio por Tienda**  
  Evaluación de la satisfacción del cliente según calificaciones promedio.

- 🚚 **Costo de Envío Promedio**  
  Análisis comparativo de los gastos promedio de envío por tienda.

---

## 📁 Datos

Los datos utilizados provienen del repositorio oficial del Challenge de Alura LATAM. Se cargan directamente desde URLs públicas en formato CSV.

---

## 📈 Visualizaciones

El análisis incluye gráficos de barras, pie charts, boxplots y otras visualizaciones interactivas generadas con `matplotlib` y `seaborn`, lo que facilita la interpretación de los datos para tomar decisiones informadas.
